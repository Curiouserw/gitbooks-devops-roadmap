# Arduino Nano (ATMega328p)

# 一、简介

参数

|                | 参数              |
| -------------- | ----------------- |
| 控制器         | ATmega328P        |
| Flash          | 32KB              |
| SRAM           | 2KB               |
| EEPROM         | 1KB               |
| 模拟I/O引脚    | 8个               |
| 数字I/O引脚    | 20个              |
| PWM通道        | 6个               |
| 时钟频率       | 16MHz             |
| ⾮稳压外部电源 | 7-15V （引脚 30） |
| 稳压外部电源   | 5V（引脚 27）     |
